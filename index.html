What is it?

Web page archiver is a gem for building web archives. It supports mht (this gem is actually based on takuya's mht gem) and html with data urls (the rename from the original mht name is just to emphasize the support for this alternative format). mht is known as mhtml that internet explorer's web page archive. this package can make web archives from local files and URI's
